### 等価のスライス式
var a [10]int<br><br>a[0:10]<br>a[:10]<br>a[0:]<br>a[:]
スライスの要素数<br>len()<br>スライスで確保しているメモリ領域の取得<br>cap()

```func main()  {
	s := []int{2,3,5,7,11,13}
	printSlice(s)

	s = s[:0]
	printSlice(s)

	s = s[:4]
	printSlice(s)

	s = s[2:]
	printSlice(s)

	s = s[1:]
	printSlice(s)

	s = s[:3]
	printSlice(s)
}

func printSlice(s []int)  {
	fmt.Printf("len=%d cap=%d %v \n", len(s), cap(s),s)

}
```
<br>結果<br>
len=6 cap=6 [2 3 5 7 11 13]<br> 
len=0 cap=6 [] <br>
len=4 cap=6 [2 3 5 7] <br>
len=2 cap=4 [5 7] <br>
len=1 cap=3 [7] <br>
len=3 cap=3 [7 11 13] <br>
[0:5]左側より下の数値は切り捨てられる

### range

```
var pow = []int{1,2,4,8,16,32,64,128}

func main()  {
	for iインデックスの数字, vインデックスの要素 := range pow {
		fmt.Printf("2**%d = %d\n",i,v)
	}
}
```
結果<br>
```
2**0 = 1
2**1 = 2
2**2 = 4
2**3 = 8
2**4 = 16
2**5 = 32
2**6 = 64
2**7 = 128
```
