### Function closures
```
func adder() func(int(5)) int {
	sum := 0
	return func(x(5) int) int {
		sum += x
		return sum
	}
}
returnで関数自体を返している。

func main()  {
	pos, neg := adder(),adder()
	for i := 0;i < 10;i++ {
		fmt.Println(
			pos(i(5)),
			neg(-2*i),
			)
	}
}
結果

0 0
1 -2
3 -6
6 -12
10 -20
15 -30
21 -42
28 -56
36 -72
45 -90
```
